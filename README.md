## 크레이지 아케이드
* 설명
```
물풍선을 이용하여 상대를 물방울에 가두어 터뜨리는 2인용 액션 게임
```
* 조작법
```
- 1P
RDFG, 좌측 Shift키, 좌측 Ctrl키

- 2P
방향키, 우측 Shift키, 우측 Ctrl키
```
* 구현 계획
```
- 캐릭터
배찌, 디지니

- 아이템
물풍선, 물줄기, 스케이트

- 맵
패트릿, 쿠키
```
